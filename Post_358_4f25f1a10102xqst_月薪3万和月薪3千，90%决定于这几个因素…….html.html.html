月薪3万和月薪3千，90%决定于这几个因素……

By: 阿萍xw 原文发布于：2017-11-19 08:35:16

为什么有的人能够拿到月薪3万，有的人只是月薪3千；有的人能过上自己想要的生活，而有的人却距离自己的目标越来越远？

主要是因为他们有以下五大特征：

第一个特征：付出就是立即要回报

1、付出就想马上有回报——你适合做钟点工

2、期望能按月得到报酬——你适合做打工族

3、耐心按年度领取年收入——是职业经理人

4、能耐心等待三到五年——适合做创业者

5、用一生的眼光去权衡——你就是企业家

这世上很多人，只愿意用钟点工的思维，去换取企业家的结果，所以会纠结，会痛苦。他们不懂得只有春天播种，秋天才会收获。


第二个特征：对新事物反映迟钝

15年前说：将来拿着手机，可以全球，对话视频，有人说是做梦。

5年前说：以后手机代替电脑可买火车票，有人说是做白日梦。

3年前说：手机可以代替取款机，可随时随地支付购物，还有人说这肯定是做梦。

1年前有人说：手机可以坐在家里，遥控万物，联通万物，有人说：纯粹胡说八道。
这个世界的变化越来越快。

从第一台计算机诞生到互联网发明，不过44年；从门户网站到电子商务的崛起，不过5年；从智能手机普及到微商遍地，不过1年！

当别人不懂的时候，他明白；当别人明白的时候，他在做；当别人在做的时候，他已经成功了！长江后浪推前浪，前浪死在沙滩上。

第三个特征：不愿意承担任何风险

三十年前，洗衣机一上市，有人说洗不干净，结果：比别人多用手洗了十年。

十五年前，太阳能热水器上市，有人说它烧不热水，结果：多花了十年电费。

十年前，很多楼房开盘上市，有人说房价一定会跌，结果：白白多干了十几年。

机会是留给有准备的人！

在一个高速发展的社会里，不去冒险就是最大的风险！

所谓的安全感，就是温水煮青蛙。唯有踏出第一步，试验各种可能，并承担一些责任，看上去有点冒险，实际上是在降低人生未来的风险。

只要不犯法，不要怕犯错！若是因为害怕失败，就连尝试的勇气都没有，表面上是逃避冒险，事实上是已经陷入最大的风险之中！

第四个特征：总幻想一夜暴富

如今的中国，上上下下都看到了同一件事：“暴利时代”已经过去了，因为“暴利时代”只存在于：产品短缺、信息不畅、需求粗放的时代，而现在这些条件都不存在了。

这就好比一个人过了身体发育阶段，接下来身高都已经定型了，唯一还在进步的只有自己的智慧，而很多人依然迷恋那个粗放发展的阶段，千方百计的想一夜暴富，注定是徒劳！

上一个阶段，是胆大的人淘汰胆小的人，而一旦社会步入完善阶段，未来将是有文化的人淘汰没有文化的人。

以前是人口红利，未来是智慧红利，这是完全不同的逻辑，未来不再有暴富，但并不代表不能致富。未来的致富是依靠智慧、知识和创新。


成功人士必备的5句话

以下是非常成功人士每天都会说的5句话，用这些激励他们采取行动，进而取得更大成就。

1. 别人都不愿做这事，那就让我来做吧

做其他人不愿做的事情：挑选一件他人不会做的事情。它可能很简单很琐碎，但这都没有关系。无论它是什么事情，去做就行了。你很快就会与其他的人略有不同。

2. 事情根本没有那么糟糕

每天做一些让你有点害怕的事情，身体或情绪上略有畏惧都可以。然后，相信自己能找到解决因此产生的任何问题的办法，因为你完全可以做到。成功之后你会得到这样的成就感：“难以相信我居然做到了！”

3. 我不关心其他人如何想

多数时候，我们应该关心其他人会怎么想。但是，在追求自己真正想要的生活方面，这条就不适用。比如：如果你非常想创办一家企业，同时担心其他人可能会说你疯了，你不妨先去把公司开起来。

找到一件你因为担心别人怎么想如何说而没尝试过的事情，勇敢地开始做。你的人生是自己的，应该用你自己的方式来打造。

4. 我应该做得更好

我们都尽了力，但我们都有些事情原本可以做得更好。成功人士从来不指望事情完美，但他们会思考如何不断进步。

因此在一天将过去时要反思，想想自己干得不错的事，然后思考可以做得更好的事，分析原因担起自己的责任，向自己承诺明天你会干得更棒。

5. 没关系，我只需比别人干得更多

正如帆船美洲杯获胜团队——甲骨文美国号的船长斯皮瑟所说：“我平生几乎没有遇到过，做得比其他人少能成为最终获胜的好策略。”

你可能没经验、没资金、没关系、没天赋，但你可以比别人想得更多、忙得更多、做得更多。

即使在诸事似乎都对你不利时，努力坚持依然能成为你的竞争优势，而且它们可能是你真正需要的唯一优势。
